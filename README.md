Trinity's World of Warcraft Packet Parser Tweaked for use with SkyFire 

WowPacketParser
========


Usage
--------------------------------------------------

* Compile WowPacketParser using Debug or Release.
* Edit "WowPacketParser.exe.config" to fit your needs.
* Drag files to the executable.
